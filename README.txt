
# Employee Compensation Forecasting Application

This project includes SQL scripts for database setup, sample CSV data for employee, roles, and locations, an Excel sheet for forecasting compensation, and documentation.

## Files Included

- `create_tables.sql`: SQL script to create database and tables.
- `Employee.csv`, `Roles.csv`, `Location.csv`: Sample datasets.
- `Forecasting.xlsx`: Sample Excel forecasting.
- `Project_Report.docx`: Project documentation.
